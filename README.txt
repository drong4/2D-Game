
2D-Sidescroller Fighting Game
Personal project that I started over my Spring break of 2017.

Short gameplay demo: https://www.youtube.com/watch?v=m6isjl7PGVM
  -Minor enemies in the beginning
  -"Boss" at the end
  -Demonstrates game mechanics included so far:
    -Enemy attacks/animations
    -Grounded/Aerial attacks
    -Counter attack
    -Dash (evade via invincible frames)
    -Shoot projectile
    -Healing (no animation or sounds though)
    -Basic UI (healthbars)

All character art/animations are made by us.

The rest of the art and the sound effects are all royalty-free.
